# SCourt 소스 정책 확인 가이드

작성일: 2026-04-08
문서 버전: v1.0

## 1. 목적
이 문서는 사법정보공개포털 기반 데이터 수집 전에 반드시 확인해야 할 정책 항목과 승인 절차를 정의한다.

대상 소스
- https://portal.scourt.go.kr/pgp/index.on

## 2. 현재 판정 상태
- source_status: policy_check
- 원문 자동 수집: 보류
- 메타데이터 수집: 제한적 허용(출처/사건번호/선고일/법원명 수준)

## 3. 보류 근거 요약
- 저작권 보호 관련 문구 및 권리자 표기 확인
- 판결서 열람/사본 제공 절차에 인증 및 수수료 절차 존재
- 자동 수집 허용 범위를 명시적으로 확인하지 못함

## 4. 승인 전 허용 범위
허용
- 사건번호, 선고일, 법원명, 판례명, 출처 URL 메타데이터 정리
- 수기 입력 또는 수동 확인 기반 목록 작성

금지
- 원문 대량 다운로드
- 우회 자동화(인증 우회, 비정상 트래픽 유발)
- 수수료/제공 절차를 우회하는 수집

## 5. 정책 확인 체크리스트
아래 항목이 모두 충족되기 전까지 source_status는 policy_check를 유지한다.

1. 저작권/이용약관 문서 원문 링크 확보
2. 자동 수집 허용 여부 명시 확인
3. 재배포/2차 가공 허용 범위 확인
4. 상업적 사용 가능 여부 확인
5. 허용 요청량(일/시간당) 확인
6. API 또는 공식 데이터 제공 채널 존재 여부 확인
7. 문의 담당 부서와 회신 근거(메일/공문) 저장

## 6. 승인 후 전환 규칙
- status를 approved로 변경하기 위한 최소 조건
  - 서면 회신 또는 공개 정책 문서 근거 1건 이상
  - 자동수집 허용 범위(대상, 빈도, 저장, 이용 목적)가 명시됨

- approved 이후에도 다음은 필수
  - 요청 빈도 제한
  - 실패/차단 로그 보존
  - 출처와 수집일 메타데이터 강제 저장

## 7. 운영 기록 위치
- 정책 체크 로그: code/data/manifests/templates/policy_check_log_template.csv 기반으로 기록
- 문의 템플릿: code/data/templates/scourt_permission_request_email.md

## 8. 체크리스트 점검 결과 (2026-04-08)

| 항목 | 결과 | 근거 | 후속 조치 |
|---|---|---|---|

| 2. 자동 수집 허용 여부 명시 확인 | 미충족 | 자동수집 허용 문구 미확인 | 메일/공문 문의 필요 |
| 3. 재배포/2차 가공 허용 범위 확인 | 미충족 | 허용 범위 문서 미확보 | 정책 문서 회신 필요 |
| 4. 상업적 사용 가능 여부 확인 | 미충족 | 상업적 이용 명시 미확인 | iOS 배포 전 필수 확인 |
| 5. 허용 요청량(일/시간당) 확인 | 미충족 | Rate limit 기준 미확인 | 기술 가이드 요청 |
| 6. API 또는 공식 데이터 제공 채널 존재 여부 확인 | 미충족 | 공개 API 문서 확인 불가 | 공식 제공 채널 문의 |
| 7. 문의 담당 부서와 회신 근거 저장 | 진행중 | 문의 템플릿 작성 완료 | 발송 후 회신 증빙 저장 |

판정
- current_status: policy_check 유지
- 원문 자동 수집: 불가(승인 전)
- 메타데이터 수집: 제한적 허용(수기/수동 확인 기반)
