[AI_SYS Stitch Prompt Pack]
버전: v1.0
언어: 한국어
용도: Stitch에 화면별 개별 업로드/입력

사용 방법
1) 아래 각 섹션을 개별 복사해서 Stitch에 붙여넣습니다.
2) 한 화면씩 생성한 뒤 같은 스타일을 유지하려면 공통 스타일 가이드를 매번 함께 붙여넣습니다.
3) 모든 텍스트는 한국어 기준입니다.

--------------------------------------------------
[공통 스타일 가이드 - 모든 화면에 함께 사용]
모바일 우선 학습 앱 UI. 서비스명 AI_SYS. 대상은 경찰 공무원 시험 준비생. 톤은 신뢰형·집중형. 네이비/딥블루 기반 + 민트 포인트. 카드형 정보 구조. 큰 터치 타겟. 상태 피드백(로딩/오류/빈 상태) 명확히. 과도한 장식 금지, 가독성 우선. 모든 텍스트는 한국어로.

--------------------------------------------------
[화면 1 - 홈(복습 대시보드)]
AI_SYS 앱의 홈 화면을 설계해줘. 대상은 경찰 공무원 시험 준비생. 앱 첫 진입 화면이며 목표는 오늘 복습 우선순위를 즉시 보여주는 것. 필수 요소: 상단 타이틀(복습 대시보드), 오늘 복습 추천 카드 3개(판례명/핵심 쟁점/우선순위 배지), 최근 오답 리스트(날짜/판례명/헷갈린 포인트), CTA 2개(복습 시작/판례 검색으로 이동), 빈 상태 UI(오답 없음 안내+시작 버튼). 디자인: 네이비·딥블루 기반 + 민트 포인트, 카드형 정보 구조, 모바일 우선, 큰 터치 타겟, 한글 가독성 높은 타이포. 모든 텍스트는 한국어로.

--------------------------------------------------
[화면 2 - 카메라/OCR 촬영]
AI_SYS 앱의 카메라/OCR 화면을 설계해줘. 문제 지문 촬영 후 텍스트를 추출해 판례 검색으로 연결하는 화면. 필수 요소: 촬영 프리뷰+가이드 프레임, 하단 CTA(촬영/취소), 촬영 가이드 문구, OCR 처리 중 상태(로딩+텍스트 추출 중), OCR 결과 미리보기(텍스트 일부), 다음 CTA(판례 검색으로 이동/직접 수정). 예외 상태: OCR 실패(재촬영/직접 입력), 카메라 권한 없음(권한 요청 배너). 디자인: 단계 전환이 명확한 UI, 시선 분산 최소화. 모든 텍스트는 한국어로.

--------------------------------------------------
[화면 3 - 판례 검색]
AI_SYS 앱의 판례 검색 화면을 설계해줘. 검색 방식은 키워드, 판례번호, OCR 텍스트 3가지. 필수 요소: 입력 영역 3종, 추천 키워드 칩(영장주의/자백배제법칙/위법수집증거), 검색 결과 카드(판례명/요약 1~2줄/관련 과목·쟁점 태그/상세 보기), 필터·정렬(관련도순/최신순). 예외 상태: 검색 결과 없음(유사 키워드 제안), 로딩 스켈레톤. 디자인: 빠른 스캔 가능한 리스트 중심, 핵심 메타정보 한눈에. 모든 텍스트는 한국어로.

--------------------------------------------------
[화면 4 - 판례 요약 상세]
AI_SYS 앱의 판례 요약 상세 화면을 설계해줘. 긴 판결문 대신 핵심 쟁점을 빠르게 이해하도록 구성. 필수 요소: 헤더(판례명/사건번호/법원·선고일), 핵심 카드 3개(쟁점/결론/시험 포인트), 출처 영역(원문 참조 정보+링크 버튼), 유사 판례 카드 리스트(판례명/짧은 요약/이동 버튼), CTA(관련 문제 보기/다른 판례 비교). 예외 상태: 출처 불완전 경고 배너+후보 판례 제시. 디자인: 중요도 순 카드 배치, 인지 부담 최소화. 모든 텍스트는 한국어로.

--------------------------------------------------
[화면 5 - 관련 문제 풀이]
AI_SYS 앱의 관련 문제 풀이 화면을 설계해줘. 목표는 판례 이해를 즉시 문제 적용으로 연결하는 것. 필수 요소: 문제 본문(객관식), 선택지 4개, 진행 상태(예: 1/5), CTA(정답 확인/오답 저장), 채점 후 피드백(정답·오답 표시/짧은 해설/연결 판례 핵심 포인트). 디자인: 시험 앱처럼 집중도 높은 레이아웃, 선택·채점 상태가 명확히 구분되는 시각 체계. 모든 텍스트는 한국어로.

--------------------------------------------------
[화면 6 - 오답 저장]
AI_SYS 앱의 오답 저장 화면을 설계해줘. 틀린 문제와 연결 판례를 기록해 다음 복습으로 이어지게 만들어줘. 필수 요소: 문제 요약/연결 판례명 표시, 입력 필드(헷갈린 쟁점/메모), 버튼(저장/취소), 저장 완료 토스트(복습 목록에 추가되었습니다). 예외 상태: 저장 실패(재시도/임시 저장), 네트워크 불안정 배너. 디자인: 입력 피로가 적은 간결한 폼, 저장 후 다음 행동(복습/홈 이동) 제안. 모든 텍스트는 한국어로.
